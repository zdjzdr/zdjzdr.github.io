Day39 Python进程池

multiprocess模块中的Pool类有以下方法：

  1. pl = Pool(5) 实例化对象，’5’ 代表5个队列
  2. pl.map() 同map方法一样,map(funcname, iterable) 默认异步的执行任务，且自带close和join
  3. pl.apply() 同步调用
  4. pl.apply_async() 异步调用 和主进程完全异步 需要手动close 和 join
